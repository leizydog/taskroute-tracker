# TaskRoute Tracker

GPS-enabled task management and route tracking application with ML-powered predictions.

## Features

- Task management with GPS location tracking
- Real-time route monitoring
- ML-powered ETA predictions
- User authentication and profiles

## Tech Stack

- **Backend:** FastAPI + PostgreSQL + PostGIS
- **Frontend:** React
- **ML:** Prophet + XGBoost
- **Deployment:** Docker

## Development Setup

### Prerequisites
- Python 3.8+
- Node.js 14+
- Docker & Docker Compose
- PostgreSQL with PostGIS extension
